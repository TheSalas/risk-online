# RISK Online

Proyecto inicial para la versión web de RISK, jugable online con amigos.

## Estructura

- `client`: Aplicación web en React.
- `server`: Backend en Node.js/Express + Socket.io (multijugador en tiempo real).

## Instalación rápida

### Frontend
```bash
cd client
npm install
npm start
```

### Backend
```bash
cd server
npm install
npm start
```
